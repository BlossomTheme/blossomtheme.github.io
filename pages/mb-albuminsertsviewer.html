    Installation

    Since this is a theme for the MusicBee plugin named AlbumInsertsViewer, it is required that you already have it installed before installing the theme. link to the plugin

    1. Open Run win + r

    2. Type %appdata and enter.

    3. This will take you to C:\Users\{USERNAME}\AppData\Roaming.
    4. Go to MusicBee directory. Destination : C:\Users\{USERNAME}}\AppData\Roaming\MusicBee
    5. Find albuminsertsviewer.colors.conf. this file should already be present if you installed the extension correctly.
    6. Replace the above file with the downloadable file in this repo with the same name.
    7. Relaunch MusicBee, Enjoy.
