    Installation

    1. Locate your KiCad configuration directory:
    2. Windows: C:\Users\[YourUsername]\AppData\Roaming\kicad\7.0\colors\ (or %APPDATA%\kicad\7.0\colors)
    3. Linux: ~/.config/kicad/7.0/colors/

    4. macOS: ~/Library/Preferences/kicad/7.0/colors/

    5. Create the colors folder if it doesn't exist:

    6. If the colors directory is not present in your KiCad 7.0 configuration folder, create it manually.

    7. Copy the theme JSON file:

    8. Place your .json theme file into the colors folder you located or created in step 1.

    9. The file name (without the .json extension) will be the name that appears in KiCad's theme list.

    10. Restart KiCad:

    11. Close all KiCad windows completely and reopen the application to ensure the new theme is detected.

    12. Select the new theme in each editor:

    13. Open the Schematic Editor, go to Preferences → Preferences → Schematic Editor → Colors
    14. In the "Theme" dropdown menu, select your newly installed theme and click OK or Apply
    15. Repeat this process for other editors:
    16. PCB Editor: Preferences → Preferences → PCB Editor → Colors
    17. Symbol Editor: Preferences → Preferences → Symbol Editor → Colors
    18. Footprint Editor: Preferences → Preferences → Footprint Editor → Colors

    Note: Each KiCad editor (Schematic, Symbol, Board, and Footprint editors) can have its own color theme selected independently.

    That's It... Enjoy!

    About BlossomTheme

    An open-source initiative based in Sri Lanka for curating vibrant color themes with accessibility in mind. Our palettes maintain a color contrast ratio of 5.06:1, ensuring readability without compromising aesthetics. Join us in blooming the world of design with thoughtfully crafted, inclusive color schemes.
